Log Types (Relative):

	Trace
	Debug
	Warning
	Information
	Error
	Fatal (Critical)

Website:
https://nlog-project.org/

Github:
https://github.com/NLog/NLog

Nuget:
https://www.nuget.org/packages/NLog/

Install-Package NLog

Create [nlog.config] file!

Right click on nlog.config -> Properties -> Copy to Output Directory -> Copy always

Activate Intelisense in nlog.config:
Download: NLog.xsd from http://www.nlog-project.org/schemas/NLog.xsd -> Put in root application folder!
